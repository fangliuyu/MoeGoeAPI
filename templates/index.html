dev模型体验:/dev

请求示例
单人模型
/{model}?text="シンクロ召喚！"
/{model}?text="シンクロ召喚！"&escape=True&type="wav"&output="test"
多人模型
/{model}/{id}?text="[ZH]你好[ZH]"
/{model}/{id}?text="[ZH]你好[ZH]"&escape=True&type="wav"&output="test"


参数 类型 必须 注释 默认值
model string 是 模型名称
(模型列表看下方)		 /
id uint 否 角色ID
(对照表看下方)		 0
text string 是 输入文本 /
escape bool 否 speaker是否转为UTF8 False
rawtext string 否 / /
type string 否 返回类型:
1."json"
2."wav"		 "json"
output string type为wav时
所必需		 请勿使用常用词,
以免数据串扰!
(建议使用QQ号)		 /


json信息
{
"code":错误代码,
"ID":角色ID,
"speaker":角色,
"audio":base64格式的音频数据,
"time":生成所花时间,
"message":其他信息,
}
{
"code":错误代码,
"errmsg":错误信息,
}


gocq调用示例
{
"type": "record",
"data": {
"file": "base64://"+bytes.decode(rsp.audio)
}
}
当type设置为wav时必须持有output参数
{
"type": "record",
"data": {
"file": "http://127.0.0.1:25565/ygo7/6?text=duel!&type=wav&output="test"
}
}
